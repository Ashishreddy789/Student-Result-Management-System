# Student-Result-Management-System🎓
A Python-based desktop application developed during my Data Science Internship at Glowlogics Solution. This tool simplifies the process of calculating academic results and automating record-keeping for educational institutions.

🚀 Features
User-Friendly GUI: Built using Tkinter with a structured grid layout for easy data entry.

Automated Calculations: Real-time processing of total marks and average percentages for students.

Excel Integration: Automatically generates a professionally formatted .xlsx report using XlsxWriter.

Validation: Includes confirmation dialogs to ensure successful data export.

🛠️ Tech Stack
Language: Python 3.12

Library (GUI): Tkinter

Library (Data Export): XlsxWriter

Data Structure: Python Dictionaries and Lists

📋 Prerequisites
Before running the application, ensure you have Python installed and the xlsxwriter library added to your environment:

Bash
pip install xlsxwriter
🖥️ How to Use
Clone the repository:

Bash
git clone https://github.com/your-username/student-marks-system.git
Navigate to the folder:

Bash
cd student-marks-system
Run the application:

Bash
python "Internship project.py"
Enter Details: Fill in the Roll No, Name, and Subject marks.

Calculate & Save: Click the Calculate button. The total and average will appear, and a file named student_results.xlsx will be created/updated in the project directory.

📂 Project Structure
Internship project.py - The main application script.

student_results.xlsx - The generated output file (generated after first run).

README.md - Project documentation.
